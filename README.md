# cleanit

cleanit is a shell script written to clean up your mess. With cleanit, you can clear and group a directory's scattered files in appropriate sub-directories.

## How to use cleanit

```sh
$ cp cleanit /usr/local/bin
$ cd DirtyDir
$ cleanit
```
